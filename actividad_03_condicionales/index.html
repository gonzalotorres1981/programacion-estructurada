Condicionales y Lógica de Programación

Programas
Evaluación de mayoría de edad
mayoromenor.c

Usa if-else para comparar una variable contra un valor constante (18). Base fundamental de la bifurcación de ejecución.

Sistema de becas — filtro estricto
beca.c

Aplica el operador && para validar simultáneamente promedio >= 85 y 0 materias reprobadas, simulando un filtrado de datos real.

Sistema de validación escolar avanzado
sistema_de_validacion.c

Usa else if para categorizar al usuario en múltiples niveles según desempeño: con beca, con reinscripción, o en situación crítica.

Número par
numero_par.c

Aplica el operador módulo (%) para verificar si el residuo entre 2 es exactamente cero.

Número impar
numero_impar.c

Variación del anterior: verifica si el residuo entre 2 es igual a 1, identificando la propiedad opuesta.

Verificación de números primos
primo.c

Combina condicionales con ciclos for. Usa una variable bandera y break para optimizar la detección de divisibilidad.

Temas aplicados
Toma de decisiones con if, else if y else.
Operadores relacionales (>, <, ==, >=) y lógicos (&&).
Aritmética de residuos con el operador módulo (%).
Variables de estado o banderas (flags).
Optimización mediante interrupción de ciclos (break).
Lectura de datos desde teclado (scanf).
