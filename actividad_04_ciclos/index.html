Ciclos y Estructuras Iterativas

Dominio de las estructuras repetitivas for, while y do-while, incluyendo menús interactivos, valores centinela y validación de entrada.

Programas
Sumatoria con límite definido
ejemplo_while.c

Solicita al usuario la cantidad exacta de números a sumar y usa un ciclo for para iterar N veces, acumulando valores.

Menú interactivo básico
ejemplo_do_while.c

Implementa do-while para garantizar que el menú se muestre al menos una vez antes de evaluar la condición de salida (opción 2).

Sumatoria con valor centinela
ejemplo_for.c

Usa while para sumar continuamente hasta detectar el valor centinela (0), demostrando ciclos de duración indefinida.

Calculadora de promedios avanzada
calculadora_de_prom.c

Integra do-while para el menú, for para calificaciones, validación de rango con retroceso de contador (--i) y casteo a float para porcentajes.

Temas aplicados
Ciclos controlados por contador (for).
Ciclos controlados por condición/centinela (while).
Ciclos de ejecución garantizada al menos una vez (do-while).
Variables acumuladoras y contadoras.
Validación de entrada de datos (Input validation).
Conversión de tipos explícita (Casting).
