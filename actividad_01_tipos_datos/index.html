Tipos de Datos

Comprensión de la gestión básica de memoria en C: tipos primitivos, modificadores y direcciones de memoria física.

Programas
Inicialización y declaración
main.c

El programa declara e inicializa variables de distintos tipos primitivos (char, int, float, double) con sus modificadores de signo y longitud.

Uso del operador de dirección
main.c

Con el operador & se accede a la ubicación exacta de cada variable en la RAM, demostrando cómo el SO asigna automáticamente una dirección de memoria.

Salida en consola
main.c

Mediante printf y especificadores de formato (%c, %d, %hd, %f) el programa imprime el valor y la dirección de memoria de cada variable.

Tipos de datos primitivos (char, int, float, double).
Modificadores de tipo (signed, unsigned, short, long).
Operador de dirección de memoria (&).
Especificadores de formato para entrada/salida.
